# Futura Website

A modern, futuristic website designed to showcase innovation and design.

**Live Site**: [https://ft2801.github.io/Futura-Website/](https://ft2801.github.io/Futura-Website/)

## Features
- **Responsive Design**: Fully responsive layout that looks great on all devices.
- **Modern Animations**: Smooth animations powered by Framer Motion.
- **SEO Optimized**: Fully optimized for search engines.

## Technologies Used
- React
- Vite
- Tailwind CSS
- Framer Motion

## Setup
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
